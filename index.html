Microsoft Corporation es una empresa de tecnología que desarrolla, otorga licencias y respalda una gama de productos de software, servicios y dispositivos.

Oracle es una marca comercial, produce hardware y software. Sus productos incluyen hardware de servidor, aplicaciones comerciales, middleware y sistemas para crear y usar bases de datos.

IBM

IBM es una compañía multinacional que integra tecnología y experiencia. Proporciona infraestructura, software (incluido Red Hat) y servicios de consultoría.
